A cor desempenha um papel fundamental no design, influenciando percepções, emoções e a estética geral de um projeto. Para utilizar cores de maneira eficaz, é essencial compreender os diferentes esquemas de cores disponíveis. Neste artigo, exploraremos os principais formatos e modelos de cores, suas características e aplicações.

---

## 🔹 **1. HEX (Hexadecimal)**

### ✅ O que é?

O formato **HEX** é amplamente utilizado em design digital e desenvolvimento web. Ele representa cores em um código de seis caracteres que combina valores de **vermelho (R), verde (G) e azul (B)** na escala hexadecimal (0 a F).

### 🎯 Exemplo:

- **Preto** → `#000000`
- **Branco** → `#FFFFFF`
- **Vermelho** → `#FF0000`

### 💡 Quando usar?

Ideal para aplicações web, UI/UX design e desenvolvimento front-end.

## 🎨 **2. RGB (Red, Green, Blue)**

### ✅ O que é?

O modelo **RGB** é baseado na mistura de **vermelho, verde e azul** para criar diferentes cores através da emissão de luz.

### 🎯 Exemplo (valores variando de 0 a 255):

- **Azul puro** → `rgb(0, 0, 255)`
- **Amarelo** → `rgb(255, 255, 0)`

### 💡 Quando usar?

Perfeito para **telas digitais**, como monitores, aplicativos e websites.

## 🖨️ **3. CMYK (Ciano, Magenta, Amarelo, Preto)**

### ✅ O que é?

Diferente do RGB, que é um modelo **aditivo**, o CMYK é **subtrativo**, sendo usado para impressão. Ele representa as cores com base na absorção de luz em superfícies físicas.

### 🎯 Exemplo (valores de 0% a 100%):

- **Azul profundo** → `cmyk(100%, 80%, 0%, 30%)`
- **Vermelho vivo** → `cmyk(0%, 100%, 100%, 0%)`

### 💡 Quando usar?

Essencial para **materiais impressos**, como revistas, panfletos e cartões de visita.

## 🎨 **4. HSL (Hue, Saturation, Lightness)**

### ✅ O que é?

O modelo **HSL** define cores com três componentes:

- **Hue (Matiz)** → A cor base (de 0° a 360°).
- **Saturation (Saturação)** → A intensidade da cor (0% a 100%).
- **Lightness (Luminosidade)** → O brilho da cor (0% a 100%).

### 🎯 Exemplo:

- **Verde vívido** → `hsl(120, 100%, 50%)`
- **Cinza neutro** → `hsl(0, 0%, 50%)`

### 💡 Quando usar?

Útil para **ajustes finos de cores**, principalmente em CSS e interfaces dinâmicas.

## 🌈 **5. OKLCH (Oklab + Lightness, Chroma, Hue)**

### ✅ O que é?

O **OKLCH** é um modelo de cores perceptual, projetado para manter a consistência visual das cores independentemente do brilho e da saturação. Ele melhora a acessibilidade e precisão das cores.

### 🎯 Exemplo:

- **Laranja vibrante** → `oklch(0.7, 0.2, 50°)`
- **Azul claro** → `oklch(0.8, 0.1, 230°)`

### 💡 Quando usar?

Ideal para **design acessível** e ajustes refinados de cores em ambientes digitais.

## 🎨 **6. Pantone**

### ✅ O que é?

O **Pantone Matching System (PMS)** é um sistema proprietário de cores amplamente utilizado na indústria gráfica e de moda. Ele garante a reprodução precisa das cores, independentemente do meio de impressão ou material.

### 🎯 Exemplo:

- **Pantone 186 C** → Vermelho
- **Pantone 286 C** → Azul Royal

### 💡 Quando usar?

Essencial para **branding, identidade visual e impressão de alta qualidade**.

## 🏆 **Conclusão**

Cada modelo de cor tem sua aplicação ideal:

| Modelo      | Aplicação Principal               |
| ----------- | --------------------------------- |
| **HEX**     | Desenvolvimento Web               |
| **RGB**     | Telas Digitais                    |
| **CMYK**    | Impressão                         |
| **HSL**     | CSS e UI Design                   |
| **OKLCH**   | Acessibilidade e Consistência     |
| **Pantone** | Branding e Impressão Profissional |

Ao compreender as diferenças entre esses esquemas, você poderá escolher a opção mais adequada para cada projeto.
